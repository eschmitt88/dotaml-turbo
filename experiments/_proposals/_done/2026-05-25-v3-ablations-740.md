# v3-ablations-740 — duration loss-form + player identity on extended data

## Where this fits

`foundation-v3-740` (2026-05-24) converged cleanly to val_auc=0.6462
@ epoch 25 — clear regression vs iso_teambias (0.6493) but not a
crash. The full v3 stack (extended cross-patch data + duration
regression + PMAE EMA-teacher + (team,team) bias + patch token +
dropped UW-SO + multitask α) doesn't compose additively. Coverage
breakdown: HIGH=0.6565, MEDIUM=0.6450, LOW=0.6364 — anonymous tail
is the binding constraint (consistent with prior).

This experiment attributes v3's regression to specific knobs and
re-opens the closed identity axis on extended data:

- **A1**: revert duration head to 8-bucket CE. Single-line change.
  Tests whether the long-pending regression switch (v3's contribution)
  was a net negative.
- **A2**: add 4M-param per-player identity embedding to the v3 stack.
  Tests whether embedding-prelim-740's NULL result was data-size-limited
  rather than an architectural truth.

## Anchor table

| Experiment | val_auc | Notes |
|---|---|---|
| v3 (primary anchor) | 0.6462 | full stack, extended data, dur regression |
| iso_pmae | 0.6464 | v3 minus most; 7.40-only |
| baseline_multitask_repro | 0.6470 | foundation-mvp baseline; 7.40-only |
| cleanup-740 | 0.6477 | engineered features + transformer, 7.40-only |
| embedding-prelim-740 | 0.6476 | + 16M player emb (NULL), 7.40-only |
| multitask-740 | 0.6495 | hand-tuned multitask + (team,team), 7.40-only |
| iso_teambias | 0.6493 | best v2 component; 7.40-only |

## Success thresholds

- **Either ablation ≥ 0.6485** = meaningful lift (within 0.0008 of iso_teambias).
- **Either ablation ≥ 0.6493** = beats iso_teambias on extended data, real win.
- **A2 ≥ 0.6485** specifically = identity axis is no longer closed; scale-up follow-up justified.

## Live monitoring

Per `~/.claude/CLAUDE.md`. Poll log every 30-45 min. Halt early on
PATTERN of 3+ consecutive epochs of: train loss increasing,
val_auc at random, NaN/Inf, GPU 0% util sustained, kernel events.

Be patient on bumpy early trajectories (v3 dipped at epochs 6-10
then recovered at 11 → late-phase 25), but a crash or NaN is
grounds for immediate halt regardless of pattern.

## Diagnostics

Both ablations write `metrics_<ablation>.json` with:

- val_auc trajectory per epoch
- val_metrics_at_best (auc, log_loss, acc, brier, dur_*, item_mAP_at_10)
- coverage_bucket_val_auc (low/medium/high terciles by n_games_log1p)
- patch_id distribution (train + val)
- delta_vs_v3, delta_vs_iso_teambias, delta_vs_embedding_prelim
- A2 only: embedding-table L2 norm distribution; cosine-similarity sample for 10 random non-anonymous players
